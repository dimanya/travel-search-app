# Travel Search App (beta)

Web-приложение, которое помогает быстро набросать маршрут поездки с помощью AI
и сразу перейти к покупке билетов и бронированию отелей по партнёрским ссылкам.

> ⚠️ Проект в **beta**: маршруты и цены примерные, все детали поездки нужно проверять вручную.

---

## Функции

- Поиск демо-рейсов (таблица на вкладке **Search**)
- Вкладка **AI Planner**:
  - ввод города отправления, количества дней, бюджета и интересов
  - генерация маршрута по дням через OpenAI (или fallback-маршрут, если API недоступен)
  - приблизительная оценка бюджета поездки
  - кнопки:
    - **Google Maps** — открыть город дня на карте
    - **Билеты** — переход на Aviasales по партнёрской ссылке
    - **Отели** — переход на Booking (или другой сервис)
