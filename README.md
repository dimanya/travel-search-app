# Travel Search App

Full-stack demo app (Node + React + MUI)

- Backend: Express on :3000
- Frontend: Vite React on :3001

## Run locally

```bash
# 1) install root tool
cd server && npm install && cd ..
cd client && npm install && cd ..
npm install

# 2) run both (backend+frontend)
npm run start
```

Open http://localhost:3001
